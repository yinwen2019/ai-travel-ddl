# 00 — 项目总览

AI 顶会信息聚合展示平台，展示 15+ AI 顶会的历年举办地、投稿 DDL、开会时间。
纯静态站点，Vue 3 + Vite + Tailwind CSS，部署至 GitHub Pages。

## 核心功能

- **会议维度视图**：以会议为列的布局，多级联动导航选取，卡片按年份降序，优先展示未开始/未截稿的会议
- **年份视图**：按年份分组的卡片网格，含多节点倒计时进度条
- **DDL 紧迫度可视化**：颜色标注（已过期灰 / ≤7天红 / ≤30天橙 / >30天绿）+ 相对天数提示
- **地点旅游百科**：城市气候、美食、景点、旅行贴士，从会议卡片渐变地点链接新标签页打开；`官网 ↗` 打开该届会议外部页面
- **AI 数据更新**：Python 脚本定期搜索更新会议数据，GitHub Actions 定时触发
- **深色模式 / 响应式**：三档断点适配桌面、平板、移动端
